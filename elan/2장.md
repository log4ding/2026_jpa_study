# 2. JPA 시작

## 5. persistence.xml 설정

- Q 여러 DB 를 사용해서 JPA 를 구성한다면 어떻게 해야할까?
  - 애노테이션에 특정 영속성 유닛 설정을 해야할지.
  - 아니면 하나의 DB 만 사용하게 하는 정책이 필요할지.
- 방언 기능과 함께 `show_sql` 을 사용한다면 정말 유용할것 같음.

## 6. 애플리케이션 개발

- 엔티티 매니저 팩토리는 생성비용이 아주 크기 떄문에 전체에서 한번만 생성하고 공유해서 사용해야함.
- 엔티티 매니저는 JPA 의 대부분의 기능을 제공하고 DB 커넥션과 밀접하게 연결되어있어서 스레드간 공유 및 재사용하면 안됨.
